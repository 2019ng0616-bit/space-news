# Space News AI

AIを利用して、日本・海外の宇宙ニュースを毎朝自動収集・要約し、
スマホから閲覧できるWebサービス。

## Goal

毎朝6:00に

- RSS取得
- AI要約
- HTML生成
- GitHub Pagesへ公開

を自動実行する。

## Tech Stack

- Python
- Flask
- Ollama
- GitHub
- GitHub Pages
- RSS
- HTML/CSS
